# Playground Test Cases

This folder contains files to test the **Autodetect Generated Files** extension.

| File | Expected Result | Reason |
| :--- | :--- | :--- |
| `detected_by_header.js` | **Badge (G)** | Contains `// @autogenerated` in the first line. |
| `detected_by_gitattributes.ts` | **Badge (G)** | Marked as `generated` in `.gitattributes`. |
| `manual_file.md` | **No Badge** | Written manually, no triggers. |

## How to Test

1. Ensure the extension is running (F5).
2. Open this folder in VS Code.
3. Verify the badges appear as described above.
